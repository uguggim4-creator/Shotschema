# 숏폼 스토리보드 가이드 (Shorts Storyboard Guide)

당신은 지금 **유튜브 쇼츠 / 틱톡 숏폼** 콘텐츠의 스토리보드를 구성하고 있습니다.
숏폼 스토리보드는 **60초 이내**, **세로형(9:16)**, **빠른 템포**가 핵심입니다.

---

## 1. 숏폼 스토리보드 기본 원칙

### 포맷 제약:
- **화면 비율**: 9:16 (세로형) — 중요 요소를 화면 중앙에 배치
- **러닝타임**: 60초 이내 (15~30초 권장)
- **씬 수**: 최소화 — 씬 전환마다 집중력 소모
- **텍스트 오버레이**: 소리 없이 시청하는 경우 고려 → 자막 공간 확보

### 숏폼의 황금 법칙:
**"Retention at every second" — 매 초마다 이탈 이유를 제거하라**

---

## 2. 씬 구성 원칙

### 오프닝 훅 씬 (0~3초):
- 즉각적인 시각적 임팩트가 있어야 함
- 인물 소개보다 "상황/문제/놀라운 사실"을 먼저 보여줄 것
- 텍스트 훅(화면 상단 자막)과 병행

### 본문 씬 (3~50초):
- 각 씬은 5~8초를 넘지 않을 것
- 정보 전달 → 빠른 컷 전환
- 패턴 인터럽트: 예상치 못한 전환으로 이탈 방지

### 마무리/CTA 씬 (마지막 5~10초):
- 루프 구조 가능: 마지막 장면이 첫 장면과 자연스럽게 연결
- 댓글/공유 유도 자막 공간 확보
- 강한 마무리 이미지 (인상적 정지 화면)

---

## 3. 숏폼 카메라/편집 스타일

| 상황 | 스타일 |
|------|--------|
| 정보 전달 | 직접 카메라 응시, 빠른 자막 |
| 스토리텔링 | 핸드헬드, 리얼리티 느낌 |
| 훅 장면 | 극단적 클로즈업 또는 와이드 전환 |
| 반전/놀라는 순간 | 빠른 줌인, 효과음 병행 |
| CTA | 정면 응시, 간결한 배경 |

---

## 4. 세로형 화면 구성 팁

- 인물은 화면 중앙에 (좌우 여백 활용 금지)
- 텍스트 오버레이는 상단 1/3과 하단 1/3 영역에 배치
- 화면 전체를 채우는 인물/사물 클로즈업이 효과적
- 배경은 최소화 — 복잡한 배경은 소형 화면에서 노이즈
