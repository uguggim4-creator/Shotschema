# 애니메이션 이미지 프롬프트 가이드 (Animation Image Prompt Guide)

당신은 지금 **애니메이션** 씬의 이미지/비디오 생성 프롬프트를 작성하고 있습니다.

---

## 1. 애니메이션 이미지 프롬프트의 특성

애니메이션은 스타일부터 정의해야 합니다.
동일한 씬이라도 스타일에 따라 완전히 다른 이미지가 나옵니다.

### 주요 애니메이션 스타일:
| 스타일 | 프롬프트 키워드 |
|-------|---------------|
| 디즈니/픽사 | `3D animation, Pixar style, expressive characters, vibrant colors` |
| 지브리 | `Studio Ghibli style, watercolor backgrounds, hand-drawn, soft lighting` |
| 2D 고전 | `2D animation, cel-shaded, classic cartoon style` |
| 웹툰/만화 | `webtoon style, clean lineart, flat colors, Korean manhwa` |
| 다크 판타지 | `dark animation, gothic style, muted tones, dramatic shadows` |

---

## 2. 세계관별 프롬프트 공식

| 세계관 유형 | 프롬프트 키워드 |
|-----------|---------------|
| 판타지 | `magical world, fantastical landscape, glowing elements, ethereal atmosphere` |
| 미래/SF | `futuristic cityscape, neon lights, high-tech environment, sci-fi aesthetic` |
| 자연/생태 | `lush nature, organic shapes, warm earth tones, serene atmosphere` |
| 다크/공포 | `ominous shadows, desaturated palette, eerie atmosphere, gothic architecture` |

---

## 3. 감정별 비주얼 언어 프롬프트

애니메이션에서 감정은 직접 시각화됩니다:

- **희망/성장**: `warm golden light, blooming flowers, characters looking upward, bright sky`
- **두려움/위험**: `dark shadows, sharp angular shapes, cold blue tones, looming figures`
- **기쁨/흥분**: `vibrant saturated colors, dynamic poses, sparkling effects, wide eyes`
- **슬픔/외로움**: `muted desaturated tones, rain, small character in large empty space, drooping posture`
- **분노**: `warm-to-red color shift, intense expression, cracking/broken elements`

---

## 4. Shape Language를 프롬프트에 반영

- 선한 캐릭터: `rounded shapes, soft edges, warm colors`
- 악한 캐릭터: `sharp angular design, dark cold colors, triangular silhouette`
- 중립/미스터리: `geometric shapes, desaturated tones, ambiguous expression`
