# 숏폼 글쓰기 이론 참고 자료
# References: TikTok/YouTube Shorts 스크립트 작성법, Hook Theory, Retention Loop 구조

## 1. 핵심 공리: 첫 1~3초가 전부다

시청자는 2~3초 안에 계속 볼지 결정한다. 이것이 숏폼 콘텐츠의 가장 중요한 법칙.

**효과적인 훅 유형:**
- **질문형**: "왜 대부분의 사람들이 이걸 모를까요?"
- **대담한 주장**: "대부분의 영상 마케팅이 실패하는 이유는 이것입니다"
- **놀라운 사실**: "80%의 광고는 처음 3초에 버려집니다"
- **패턴 인터럽트**: 예상을 깨는 시각, 소리, 문장 중간 시작
- **감정 트리거**: 놀람, 공포, 기쁨으로 시작
- **직접 호명**: "프리랜서라면 이건 꼭 보세요"

---

## 2. 숏폼 스크립트 공식

### 공식 A: 3막 구조 (Hook → Value → CTA)
- **Hook (0~3초)**: 즉각적 주의 집중
- **Value (3~20초)**: 핵심 메시지 전달, 문제 정의, 해결책 또는 정보 제공
- **CTA (마지막 5~10초)**: 댓글, 팔로우, 링크 클릭 유도. 자연스럽게, 강요하지 말 것

### 공식 B: Problem → Agitate → Solution (PAS)
- **Problem**: 타깃이 겪는 문제를 즉시 제시
- **Agitate**: 그 문제의 고통을 구체화, 확대
- **Solution**: 제품/팁/정보를 해결책으로 제시

### 공식 C: Delayed Value (지연된 가치)
- 영상 초반에 "마지막에 알려드릴게요"를 예고
- 시청자가 끝까지 보게 만드는 구조
- 알고리즘 신호: 완주율이 높으면 추천 확률 상승

---

## 3. Retention Loop 구조

영상의 끝이 자연스럽게 시작으로 연결되어 무한 반복을 유도하는 구조.

**만드는 법:**
1. 결말을 먼저 쓰고 거꾸로 시작 짜기
2. 마지막 화면/사운드가 첫 화면과 유사하도록 매칭
3. 25초 이내 짧을수록 루프 효과 극대화
4. 훅의 단어를 마지막 줄에 다시 등장시켜 완결감 제공
5. 루프 영상에서는 CTA를 자막/댓글로 넣기 (화면 내 CTA는 루프를 방해)

**효과**: 조회수 카운트 증가, 완주율(Completion Rate) 상승, 알고리즘 노출 확대

---

## 4. 스크립트 작성 가이드라인

- **대화체**: 전문 용어 없이, "당신"과 "우리"를 사용하라
- **간결함**: 30초 영상 = 약 75~85 단어. 불필요한 말 모두 제거
- **속도 조절**: 너무 빠르면 이해 불가, 너무 느리면 이탈. 템포 테스트
- **시각 보조**: 빠른 컷, 텍스트 오버레이, 자막 (소리 없이 보는 사람 고려)
- **한 가지 메시지**: 여러 주제는 절대 금지. 하나를 깊게

---

## 5. 알고리즘과 글쓰기의 관계

| 알고리즘 신호 | 연관 글쓰기 기법 |
|-------------|----------------|
| 완주율(Completion Rate) | Delayed Value, Retention Loop |
| 재시청률 | Loop 구조, 반전 엔딩 |
| 공유/저장 | 강한 정보 가치, 감정적 공명 |
| 댓글 유도 | 질문형 훅, 논쟁 유발 주장 |

---

## 6. 숏폼 콘텐츠 유형별 공식

| 유형 | 구조 | 특징 |
|------|------|------|
| 정보 제공 | Hook → 리스트 → 요약 | 빠른 컷, 번호 활용 |
| 스토리텔링 | 훅 → 갈등 → 해결 | 감정 이입 유도 |
| Before&After | 문제 → 비포 → 애프터 | 변화의 극적 대비 |
| 반전형 | 예상 setup → 반전 → CTA | 재시청 유도 |
| 루프형 | 결말 = 시작 | 무한 반복, 중독성 |
